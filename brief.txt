QDay Prize Submission Brief
===========================

10-bit is the largest ECC key broken on real quantum hardware as of April 2026.

Project
-------
Hardware-verified ECDLP key recovery using Shor's algorithm on IBM Quantum.

Contact
-------
Aaron@vexaai.app

What is demonstrated
--------------------
1) Real IBM hardware runs with verifiable job IDs
2) Correct private key recovery and classical verification d*G = Q
3) Scaling evidence beyond baseline small-key runs

Methods summary
---------------
- Uses Shor's algorithm for ECDLP.
- Oracle computes f(a,b) = aP + b·d·P mod n.
- IQFT extracts d via d_cand = -a·b⁻¹ mod n.
- EC verification confirms d·G = Q.

Backend
-------
IBM Torino via qiskit-ibm-runtime

Limitations
-----------
- Low hit rates at larger bit sizes
- High shot counts required
- Scaling cost rises with circuit depth
